Resumen Ejecutivo

HDK es un clasificador de texto implementado en FPGA Artix-7 XC7A200T con cero multiplicaciones de matriz. A diferencia de aceleradores tradicionales que usan DSPs y MACs, HDK usa solo lógica LUT — proyección aleatoria binaria + L-BFGS — alcanzando 71.09% de precisión en 20 Newsgroups con cero DSPs, cero BRAMs y cero MatMul.

Pipeline de Inferencia

  1. Host → UART: Vector HD (20K bits) recibido a 10 Mbaud
  2. XOR Array: XOR contra 20 prototipos en paralelo
  3. POPCOUNT Trees: 20 contadores de distancia Hamming (15 niveles pipeline)
  4. Argmin: Busca la distancia mínima → clase predecida
  5. UART → Host: Clase + confianza enviados de vuelta

Executive Summary

HDK is a zero-matrix-multiply text classifier on Artix-7 XC7A200T FPGA. Unlike traditional accelerators using DSPs and MACs, HDK uses only LUT logic — random binary projection + L-BFGS — achieving 71.09% accuracy on 20 Newsgroups with zero DSPs, zero BRAMs, and zero matrix multiplications.

Inference Pipeline

  1. Host → UART: 20K-bit HD vector received at 10 Mbaud
  2. XOR Array: XOR against 20 prototypes in parallel
  3. POPCOUNT Trees: 20 Hamming distance counters (15-level pipeline)
  4. Argmin: Sequential minimum search → predicted class
  5. UART → Host: Class ID + confidence sent back
